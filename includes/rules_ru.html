Цель игры

Цель игры – как можно быстрее вспомнить слова и максимально заполнить башни с выбранной сквозной буквой. Сквозная буква задаётся автоматически и выбирается случайным образом. Слова заполняются только в строках по горизонтали.

Начало игры

Башни бывают 2х размеров:
  • «3-4-5» – башня состоит из 3х трёхбуквенных слов, 4х четырёхбуквенных и 5ти пятибуквенных слов;
  • «4-5-6» – башня состоит из 4х четырёхбуквенных, 5ти пятибуквенных слов и 6ть шестибуквенных слов.
Кроме размера башен в настройках можно выбрать язык для игры, установить уровень сложности, количество башен и ограничение по времени. В игре предусмотрены следующие уровни сложности:
  • 1 (легкий) – слова могут повторяться; нет проверки на часть речи (разрешён ввод не только существительных). При включенном ограничении по времени на заполнение башен даётся больше, чем на других уровнях.
  • 2 (средний) – слова могут повторяться; разрешен только ввод существительных в именительном падеже единственном числе;
  • 3 (сложный) – слова не могут повторяться; разрешен только ввод существительных в именительном падеже единственном числе и времени на заполнение башен, при включенном ограничении по времени меньше, чем на других уровнях.
Кол-во башен влияет на кол-во одновременно заполняемых башен. Чем больше башен, тем сложнее их заполнять на 3м уровне при запрете повторов. При включенном ограничении по времени время на заполнении башен ограничено, при этом, чем выше уровень, тем меньше времени на заполнение башни.

Подсказки

Возможность подсказок появляется только при включенной настройке «Подсказки». Воспользоваться подсказкой можно находясь на пустой строке с незаполненным словом. При первом нажатии на кнопку открывается любая буква случайным образом, кроме первой. При втором нажатии на кнопку «Подсказка» открывается ещё одна любая буква. При третьем нажатии на кнопку полностью заполняется слово. Баллы при расчете результатов за такое слово не начисляются.

Завершение игры

Завершение игры происходит в следующих случаях:
  • заполнены все слова во всех игровых полях;
  • истекло игровое время для режима с ограничением по времени;
  • нажата кнопка «Завершить игру».
